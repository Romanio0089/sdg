# SmartDragonGame

Bienvenue sur le repo de SmartDragonGame.

Ici vous pouvez trélécharger SmartDragonGame en allant sur la section Releases.           
Le jeu est disponible pour Windows, macOS, Linux et Android (Android en version BETA)

# Configuration minimale

Windows : Windows 7 et supérieur. (Pas testé sur les versions inférieures)                      
Mac OS X 10.9 minimum.                                                              
Android Kitkat 4.4 minimum.
